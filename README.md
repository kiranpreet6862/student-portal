🎯 PROJECT TITLE

Student Mental Health & Academic Pressure Analytics Platform

🧠 1. PROJECT OVERVIEW

This is a web-based platform designed to monitor and analyze student mental health.

It focuses on how academic pressure affects wellness.

The system collects daily student data and converts it into insights, reports, and suggestions.

🎯 2. MAIN OBJECTIVE

Help students manage stress and workload effectively

Provide data-driven insights instead of guesswork

Detect burnout risk early

Improve study-life balance

⚙️ 3. PROBLEM STATEMENT

Students face:

High academic pressure

Poor sleep cycles

Irregular study patterns

Increased stress & burnout

👉 Currently, there is no simple system to track and analyze these factors together.

💡 4. SOLUTION (WHAT YOUR PROJECT DOES)

Collects daily data like:

Study hours

Stress level

Sleep hours

Assignment workload

Analyzes this data using logic

Displays:

Wellness score

Stress trends

Smart suggestions

🧩 5. KEY FEATURES
🔹 Dashboard

Shows overall wellness score

Displays charts (study vs stress, sleep vs mood)

Highlights burnout alerts

🔹 Daily Check-in

Student enters:

Learning hours

Stress level

Sleep

Mood

System calculates live wellness score

🔹 Assignment Pressure

Tracks assignments with:

Deadline

Difficulty

Time required

Shows total workload

🔹 Focus Analytics

Tracks:

Deep work hours

Distracted time

Identifies best study time

🔹 Wellness Report

Generates:

Recovery score

Workload score

Balance score

Monthly comparison

🔹 Smart Suggestions

Gives recommendations like:

“Take a break”

“Improve sleep”

“Reduce workload”


🏗️ 6. TECHNOLOGY USED
Frontend:

HTML, CSS, JavaScript

ApexChart.js (graphs)

Backend:

Django (data handling + API)

Database:

SQLite (initial)

🔄 7. WORKING FLOW

Student fills daily check-in

Data stored in database

Backend processes data

Frontend displays:

Charts

Scores

Suggestions

📊 8. OUTPUT

Visual dashboard

Graphs and reports

Personalized insights

Alerts for high stress

🌟 9. UNIQUE POINTS (VERY IMPORTANT FOR TEACHER)

Combines mental health + academics (rare combination)

Uses data analytics approach, not just CRUD

Real-world EdTech use case

Scalable to mobile app 

Helps in preventive mental health care

🚀 10. FUTURE SCOPE


Integration with wearable devices
